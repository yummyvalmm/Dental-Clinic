# Mobile UI Standard Checklist (2026)

## 1. Visual & Glassmorphism Core

- [ ] **The "Golden" Glass Ratio**: Background is `rgba(255, 255, 255, 0.15)` with a 12px to 20px backdrop-blur.

- [ ] **Layered Depth**: Use Z-index to ensure "Glass" components have different blur levels (Higher = More Blur).

- [ ] **Tint Layer**: Every glass component has a 5% white "inner tint" to ensure text legibility.

- [ ] **Hardware Acceleration**: All CSS glass components use `transform: translateZ(0)` to prevent mobile lag.

## 2. Ergonomics & Touch

- [ ] **The Thumb Zone**: All primary navigation and "Submit" buttons are located in the bottom 33% of the screen.

- [ ] **Safe Touch Targets**: Every button/link is at least 44px tall and 44px wide.

- [ ] **Haptic Feedback**: Interactive elements trigger a light haptic impact on mobile tap.

## 3. State & Feedback

- [ ] **Skeleton States**: Viewports show shimmering "skeleton" placeholders during data fetch (No generic spinners).

- [ ] **Active States**: Buttons show a visual "press" effect (0.95 scale or color shift) when touched.

- [ ] **Empty States**: Every list view has a dedicated "Empty State" illustration/text if no data is present.

## 4. Accessibility & Performance

- [ ] **Contrast Check**: Text-to-Glass contrast ratio is at least 4.5:1.

- [ ] **Reduced Motion**: CSS `@media (prefers-reduced-motion)` is implemented to disable heavy blurs/animations if the user requests it.

- [ ] **PWA Manifest**: `theme_color` and `background_color` are set in manifest.json to match the Glass UI theme.

## Implementation Guidelines

### Glass Component Standards
```css
.glass-component {
  background: rgba(255, 255, 255, 0.15);
  backdrop-filter: blur(12px) saturate(180%);
  -webkit-backdrop-filter: blur(12px) saturate(180%);
  transform: translateZ(0);
  will-change: transform;
}

.glass-tint::before {
  content: '';
  position: absolute;
  inset: 0;
  background: rgba(255, 255, 255, 0.05);
  pointer-events: none;
}
```

### Touch Target Standards
```css
.touch-target {
  min-width: 44px;
  min-height: 44px;
  display: flex;
  align-items: center;
  justify-content: center;
}
```

### Active State Standards
```css
.interactive:active {
  transform: scale(0.95);
  transition: transform 0.1s ease-out;
}
```

### Reduced Motion Support
```css
@media (prefers-reduced-motion: reduce) {
  * {
    animation-duration: 0.01ms !important;
    animation-iteration-count: 1 !important;
    transition-duration: 0.01ms !important;
    backdrop-filter: none !important;
  }
}
```

## Strictly Follow These Rules

1. **Always** use the golden glass ratio for backgrounds
2. **Always** include tint layers for legibility
3. **Always** ensure 44px minimum touch targets
4. **Always** add hardware acceleration to glass components
5. **Always** implement skeleton loading states
6. **Always** provide empty states for lists
7. **Always** support reduced motion preferences
8. **Always** add haptic feedback to interactive elements
